# TravelMeet
## Social media web application for meeting locals, and searching events in the targeted area on the map.
Tech Stack: 
- Node
- Express
- Vanilla JS
- Socket.IO
- Mapbox GL JS
- Ticketmaster API
- Google API's
- Redis
- Prisma ORM
- MySQL
- Tailwind CSS
- AWS S3 Object Storage
- Passport.js
- Swiper API
- EJS
- Multer NPM Module
- Sharp NPM Module
